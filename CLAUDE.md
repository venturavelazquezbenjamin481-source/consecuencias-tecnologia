# Proyecto: Consecuencias del uso de tecnología

Página web educativa sobre las consecuencias físicas, mentales,
sociales y ambientales del uso de la tecnología.

## Comandos útiles
- Abrir en navegador: abre index.html directamente en el browser
- No hay build system — es HTML + CSS + JS puro, sin Node

## Estructura del proyecto
- index.html          — página principal / hero
- salud-fisica.html   — sección de consecuencias físicas
- salud-mental.html   — sección de salud mental
- sociedad.html       — sección de impacto social
- ambiente.html       — sección de medio ambiente
- soluciones.html     — sección de soluciones
- css/styles.css      — estilos globales compartidos
- js/main.js          — scripts de interactividad
- assets/             — imágenes y recursos

## Estilo de código
- HTML5 semántico: usar <section>, <article>, <nav>, <header>, <footer>
- CSS: variables en :root para colores y tipografía, sin frameworks externos
- JS: vanilla JavaScript, sin librerías externas
- Indentación: 2 espacios
- Nombres de clases en español y kebab-case (ej: .seccion-hero, .tarjeta-dato)
- Comentarios en español

## Convenciones de contenido
- Idioma: español (México)
- Tono: educativo, claro, dirigido a jóvenes estudiantes
- Cada sección debe tener: introducción, datos clave, explicación y soluciones
- Las estadísticas deben incluir su fuente en un atributo data-fuente=""

## Equipo
- Cada integrante trabaja en su propio archivo .html de sección
- No modificar css/styles.css sin avisar al equipo
- Nombrar las imágenes descriptivamente (ej: sedentarismo-joven.jpg)

## Lo que NO hacer
- No usar frameworks (sin Bootstrap, Tailwind, React, etc.)
- No usar imágenes sin derechos de autor
- No dejar comentarios tipo "TODO" sin resolver antes de entregar
